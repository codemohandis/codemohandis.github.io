# Mohammad Mehdi - Systems Architect & Automation Engineer Portfolio

Welcome to the source code of my personal portfolio! This project is a modern, highly responsive, and interactive single-page application built to showcase my expertise in systems architecture, AI-driven automation, and data engineering.

## 🚀 Live Demo
**[View Live Portfolio](https://codemohandis.github.io/)**

## 🛠️ Tech Stack
This portfolio is built as a **pure static HTML file** for maximum performance and zero-configuration deployment:
- **HTML5**
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (via CDN)
- **Icons:** [Lucide Icons](https://lucide.dev/) (via CDN)
- **Animations & Logic:** Vanilla JavaScript

## ✨ Key Features
- **Zero Build Step:** No Node.js, no Webpack, no Vite required. Just open `index.html` in your browser.
- **Deep Dark Mode Theme:** Custom `slate-950` background with "Neon Emerald" and "Cyber Blue" accents.
- **Agentic UI/UX:** Matrix-style grid backgrounds, glowing radial gradients, and typing animations.
- **Pipeline Architecture Cards:** Custom UI components designed to visually represent Input → Process → Output workflows.
- **Fully Responsive:** Optimized for both mobile and desktop viewing.

## 💻 Deployment to GitHub Pages

Because this is a pure static website, deploying to GitHub Pages takes 10 seconds:

1. Upload `index.html` and the `public/` folder to your GitHub repository.
2. Go to your repository **Settings** > **Pages**.
3. Under **Build and deployment**, set the Source to **Deploy from a branch**.
4. Select the `main` branch and click **Save**.
5. Your site will be live in 1 minute!

## 📄 Adding Your Resume
To make the "Download Resume" button work, simply place your PDF file in the `public/` directory and name it `resume.pdf`.
